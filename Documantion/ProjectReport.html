Executive Summary

Digitizing Logistics for the Modern Era

The GoParcel Logistics Management System is a comprehensive full-stack web application designed to revolutionize how small-to-medium courier services operate. By replacing fragmented, manual, and paper-based processes with a centralized digital platform, GoParcel automates critical tasks such as shipment booking, dynamic pricing, and real-time status tracking.

The Industry Challenge

Logistics companies utilizing traditional methods face significant operational bottlenecks. Manual data entry is slow and prone to human error, particularly in complex pricing calculations involving weight, distance, and taxes.

Furthermore, without a digital tracking system, customer transparency is non-existent, leading to high support volumes. Invoicing remains a tedious manual task, delaying revenue collection and financial reporting.

The GoParcel Solution

GoParcel introduces an Automated Booking Engine that instantly calculates costs with zero error margin. It provides a Live Tracking Dashboard that offers stakeholders real-time visibility into shipment status.

The system also features an Auto-Invoicing Module that generates professional, tax-compliant A4 invoices instantly, streamlining the financial workflow and enhancing brand professionalism.

System Architecture

Built on a robust Python-Flask foundation with a modern Glassmorphism frontend.

Frontend

HTML5 / CSS3 / JS

Utilizes semantic HTML5 for structure and advanced CSS3 features (Backdrop Filter, Flexbox, Grid) to create the signature "Glassmorphism" aesthetic. Vanilla JavaScript (ES6+) handles all client-side logic and DOM manipulation.

Backend

Python Flask

A lightweight WSGI web application framework. Flask routes API requests, handles business logic, and serves the frontend. It was chosen for its simplicity, scalability, and ease of integration with SQLite.

Database

SQLite3

A serverless, self-contained SQL database engine. It provides zero-configuration storage, making the application portable and easy to deploy without complex database server setups.

Communication

AJAX & JSON

Data is exchanged asynchronously between the frontend and backend using the Fetch API and JSON. This ensures a smooth Single Page Application (SPA) feel without page reloads.

Installation & Setup

1. Prerequisites

  • Python 3.x installed on your system.
  • Pip (Python Package Installer).

2. Install Dependencies

pip install flask

3. Initialize Database

Run the database script to create the required tables and schema file.

python Database.py

4. Run Application

python App.py

5. Access Portal

Open your browser and navigate to: http://127.0.0.1:5000
